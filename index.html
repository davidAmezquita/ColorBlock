ColorBlock

Instructions: Game starts with multiple blocks of different colors. Your job is to get all the blocks to be the same color before you run out of moves. In order to do that, you must click on a block that is adjecent to blocks chained together. Chained blocks are those who are adjecent to eachother and have the same color. The top left most block is root to the chain so any block adjecent to it can be chained.

Difficulty-> Easy
